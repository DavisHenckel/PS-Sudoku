# GenerateGrid

## SYNOPSIS
Generates an Empty 2D Arraylist

## SYNTAX

```
GenerateGrid [[-Difficulty] <String>] [<CommonParameters>]
```

## DESCRIPTION
Generates an Empty 2D Arraylist that will be used for the Sudoku Board

## EXAMPLES

### EXAMPLE 1
```
$SudokuBoard = GenerateGrid
```
